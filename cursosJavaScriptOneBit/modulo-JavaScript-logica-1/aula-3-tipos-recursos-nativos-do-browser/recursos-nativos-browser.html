Recursos Nativos do Browser > Utlizando o console do navegador

Tipos de console

  • console.log("") -> para impressão no console
  • console.info("") -> gera string com letras azuis
  • console.warn("") -> strings amarelas
  • console.error("") -> string de erro.

Tipos - Alert > Confirm > Prompt

  • alert("") -> exibe uma caixa de mensagem para o usuário
  • confirm("") -> alert com botão de cancel e ok ("confirmar ou não uma msg")
  • prompt("Qual seu nome, piloto?") -> O prompt, exibe a msg + um campo de input + btn de cancel e ok
