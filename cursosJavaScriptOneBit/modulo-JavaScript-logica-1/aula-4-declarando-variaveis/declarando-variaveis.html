Declarando Variaveis > Utlizando o console do navegador

let

  • let spaceShip = "FE supernova" -> declaração de variavel
  • spaceShip = "dando um novo valor" -> atribuindo um novo valor

Const

  • const squad = "estelar" -> declaração de constante
  • squad = "espacial" -> como é uma constante ele n recebe atribuições, inclusive irá gerar erro.

Var

  • Funciona como o let
  • Qual a diferença de um do outro? Ele possui um motivo que será tratado futuramente

Obs:

  • let semvalor = noValue -> Irá gerar uma string undefined
  • let depatureConfirm = confirm("nave p/ p/ decolar?") -> atribui o valor de um retorno
  • let depatureConfirm = prompt("qual a nave?") -> atribui o valor de um retorno
