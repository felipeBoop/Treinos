Comparacoes de Dados

o Igual "=="

  • console.log(velocity == 80) -> comparamos com os dois iguais, pois um só significa atribuição.
  • console.log(spaceship == "golias")
  • console.log(velocity == "80") -> irá retornar true, devido a sua tipagem fraca.

A igualdade "==="

  • console.log(velocity === "80") -> ira retornar false, pois com === iguais, ele irá comparar o tipo também

O diferente "!="

  • console.log(velocity != 70) -> ira retornar true, pois são valores diferentes
  • console.log(velocity !== 70) -> mesma coisa que o igual com ===

Maior que ">"

  • console.log(velocity > 70) -> ira retornar true, pois velocity é 80
  • console.log(spaceship > "helmet") -> irá verificar se uma string vem antes ou depois da outra, exemplo "golias" vem antes de "helmet", pois g vem antes de h, logo retornaria false

Maior ou igual ">="

  • console.log(velocity >= 80) -> ira retornar true.
  • console.log(spaceship >= "Golias") -> verdadeiro, pois um n vem antes do outros, são iguais

Menor "<"

  • mesma lógica que o maior só que ao contrário
